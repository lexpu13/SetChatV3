# SetChat V3

Version ultra réaliste orientée accessoire de tournage.

## Ce qui a été amélioré
- design plus proche d'une messagerie moderne
- avatars plus lisibles
- conversations plus crédibles
- bulles plus réalistes
- faux indicateurs d'envoi / lecture
- création de discussion privée ou groupe
- mode PREPA / TOURNAGE

## Ouvrir le projet
1. Dézippez l'archive.
2. Ouvrez `SetChatV3` dans Android Studio.
3. Laissez Gradle synchroniser.
4. Lancez l'application.

## Limite honnête
C'est une V3 visuelle plus crédible, mais pas encore une reproduction pixel-perfect d'une app commerciale.
